Bài 1: Viết hàm nhập và xuất mảng số nguyên dùng chỉ số mảng arr[i].

Bài 2: Viết lại Bài 1 nhưng dùng con trỏ *(arr + i) để truy cập phần tử.

Bài 3: Viết hàm tính tổng mảng sử dụng con trỏ, duyệt mảng bằng cách tăng con trỏ.

Bài 4: Viết hàm tìm giá trị lớn nhất trong mảng, trả về con trỏ trỏ đến phần tử lớn nhất.

Bài 5: Viết hàm đảo ngược mảng (ví dụ: {1,2,3} → {3,2,1}) sử dụng hai con trỏ (đầu và cuối).
